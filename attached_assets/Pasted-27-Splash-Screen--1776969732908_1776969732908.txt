27. Экран выбора языка

· Перед Splash Screen'ом (или сразу после него, перед авторизацией) добавь экран выбора языка.
· Минималистичный дизайн в тёмно-синей гамме: логотип «⚡ ОНИКС» вверху, надпись «Выберите язык / Choose language» по центру.
· Кнопки выбора:
  · 🇷🇺 Русский
  · 🇬🇧 English
· Выбранный язык сохраняется в localStorage (ключ onix_language).
· Все последующие экраны (Splash, Авторизация, Онбординг, Дашборд, все разделы) должны использовать выбранный язык.
· Создай файл с переводами (locales/ru.json, locales/en.json) и хук useTranslation, который подхватывает текущий язык из localStorage.
· Переведи на английский все тексты интерфейса: меню, карточки, напоминания, историю, заказы, памятку, диагностику, панель механика, экран авторизации, онбординг, политику конфиденциальности.
· В настройках (Ещё → Настройки) добавь пункт «Язык / Language» для смены в любой момент.

28. Интеграция с VIN-декодером и отображение модели авто

· Сделай так, чтобы приложение определяло модель, кузов и изображение автомобиля по VIN-коду. Используй бесплатный государственный API: vPIC (vpic.nhtsa.dot.gov). Не используй платных провайдеров.
· Сервис для расшифровки VIN:
  · Создай файл services/vinDecoder.js.
  · Функция decodeVin(vin) отправляет GET-запрос на https://vpic.nhtsa.dot.gov/api/vehicles/DecodeVin/{vin}?format=json.
  · Извлеки из ответа поля: Make (марка), Model (модель), ModelYear (год), BodyClass (тип кузова).
· Локальная библиотека изображений:
  · Создай папку assets/cars/ с заглушками «вид сверху»:
    · skoda_octavia.png — для Skoda.
    · audi_a6.png — для Audi.
    · bmw_x5.png — для BMW.
  · Создай services/carLibrary.js с функцией getCarImage(make, model), возвращающей путь к изображению.
· Отображение на главном экране:
  · При входе автоматически вызывай decodeVin для VIN-кода пользователя (тестовые 0000, 2222, 3333).
  · На главном дашборде в шапке показывай название модели, тип кузова и изображение из библиотеки.
  · При ошибке определения VIN или недоступности API показывай стандартную заглушку.
